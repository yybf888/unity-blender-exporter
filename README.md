# Unity Blender Exporter

一个用于 **Blender → Unity** 的自动化 FBX 导出插件。

## ✨ 功能特性
- 一键批量导出 FBX
- 自动应用变换（避免 Unity 缩放 100）
- 支持 Unity 坐标系 (-Z Forward / Y Up)
- 参数持久化（随 .blend 保存）
- 面向游戏美术工作流设计

## 📦 安装方法
1. 下载本仓库并解压
2. Blender → Edit → Preferences → Add-ons → Install
3. 选择 `unity_exporter` 文件夹
4. 启用插件

## 📍 使用位置
`3D Viewport → N 键 → Unity Tools`

## 🧠 适用场景
- 游戏场景建模
- Unity 资源管线
- 毕设 / 教学 / 工具开发
